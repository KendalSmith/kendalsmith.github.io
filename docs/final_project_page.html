Overview

Research Questions

The aim for this project was to identify possible semantic trends across masculine, feminine, and neuter nouns in Sanskrit.
Group similarity comparisons between genders as a whole, as well as semantic trends within each gender individually were assessed.

Background

What is it?

Sanskrit is an Ancient Indic language from the Indo-European Language family. It was used as a liturgical language to pass oral traditions and despite being a sacred language it is still spoken by over 3,000,000 people throughout India today. Like its sister languages Greek and Latin, Sanskrit is a language with three grammatical genders, masculine, feminine, and neuter. This project analyses semantic likeness and gender distribution across semantic categories using 239 nouns sourced from the index of The Cambridge Introduction To Sanskrit A.M.Ruppel.


To assess gender similarity between each group of nouns, Wu-Palmer similarity scores were used through WordNet to compare the total semantic likeness between each gender. Using a hierarchical system to map semantic likeness, these scores calculate how close a pair or words is to each other. To assess semantic category distribution, nouns in the corpus were manually tagged with one of eight semantic categories. This data was then organized into bar charts.

Semantic Distribution Results

Masculine Semantic Distribution

Things of Note

  • High percentage of power terms
  • Low percentage of positive terms
  • Nature, negative, and animal relatively similar
  • Weapons are almost all masculine

Feminine Semantic Distribution

Things of Note

  • High percentage of positive terms
  • High percentage of nuetral terms
  • Body parts tended to be abstract when feminine

Nueter Semantic Distribution

Things of Note

  • High percentage of positive terms
  • Significantly highest percentage of body part terms
  • Only two weapon terms, bow (m/n), weapon (n)
